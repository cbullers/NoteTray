NoteTray

A lightweight note-taking app that lives in your Windows system tray. Summon it instantly, jot something down, and get back to work.

Features

⚡

Instant Access

Global hotkey summons your notes from anywhere. No window switching, no hunting through the taskbar.

📁

Folder Organization

Group notes into folders. Switch between them with a quick dropdown. Keep work and personal notes separate.

📝

Markdown Support

Write in plain text with markdown formatting. Toggle a live preview to see your notes rendered.

🔍

Quick Search

Filter your notes as you type. Find what you need without scrolling through a long list.

🎨

Color Coding

Assign colors to notes for quick visual identification. Important notes stand out at a glance.

🖥

Multi-Monitor

The grab bar tracks your active monitor. Your notes panel always appears on the screen you're working on.

📂

Plain File Storage

Notes stored as .md files in your AppData folder. No database, no cloud dependency, fully portable.

🌙

Dark Theme

A polished dark interface that's easy on the eyes during late-night work sessions.

🔨

Configurable

Custom hotkeys, panel width, animation speed, grab bar position, and startup behavior. Make it yours.

Quick Start

# Clone the repo
git clone https://github.com/cbullers/NoteTray.git
cd NoteTray

# Build and run
dotnet run --project NoteTray/NoteTray.csproj

# NoteTray appears in your system tray
